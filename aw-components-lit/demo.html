Technical Implementation Summary

✅ AW Naming Conventions

  • All components use aw-* prefix
  • Props use snake_case naming
  • CSS custom properties use --aw-* prefix
  • BEM classes follow aw-component__element--modifier pattern

✅ BEM + ITCSS Methodology

  • Settings: Design tokens and variables
  • Generic: Normalize and reset styles
  • Elements: Base HTML element styling
  • Objects: Layout patterns and constructs
  • Components: UI component styles
  • Utilities: Helper classes

✅ Vanilla Motion Integration

  • @motionone/dom replaces Framer Motion
  • Staggered animations for text reveals
  • Intersection Observer for scroll triggers
  • Smooth easing curves matching source

✅ Visual Style Matching

  • Color palette matches source theme system
  • Typography scales preserved
  • Spacing system consistent
  • Animation timings replicated
